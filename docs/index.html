WiFi Deauth Attack Tool

ESP32 tabanlı, OLED ekranlı ve rotary encoder kontrollü taşınabilir WiFi ağ analiz ve güvenlik test cihazı.

ESP32 C++ / Arduino I2C OLED 128×64 Rotary Encoder 802.11 Deauth Donanım Kesmesi
Özellikler
📡

Ağ Tarama

Çevredeki tüm WiFi ağlarını SSID ve RSSI ile listeler (maks. 20 ağ).

🖥️

OLED Arayüz

SSD1306 128×64 I2C ekranda seçim vurgulu ağ listesi gösterimi.

🎛️

Rotary Encoder

Fiziksel döner kodlayıcı ile sezgisel ağ seçimi ve menü navigasyonu.

⚡

Donanım Kesmesi

Interrupt tabanlı buton yönetimi ve yazılımsal debounce (500 ms).

🔬

Deauth Test

802.11 deauthentication paketi tabanlı bağlantı dayanıklılığı testi.

🌐

Tarayıcı Simülatörü

Donanım olmadan arayüzü test eden tam özellikli HTML/JS demo.

Devre Şeması & Pin Bağlantıları
  ESP32 DevKit
  ┌─────────────────────────────────────────────┐
  │                                             │
  │   SSD1306 OLED 128×64 (I2C)                 │
  │   GPIO21 (SDA) ──────── SDA                 │
  │   GPIO22 (SCL) ──────── SCL                 │
  │   3.3V  ────────────── VCC    I2C Adr: 0x3C │
  │   GND   ────────────── GND                  │
  │                                             │
  │   Rotary Encoder                            │
  │   GPIO18 ───── CLK (A)                      │
  │   GPIO19 ───── DT  (B)                      │
  │   GPIO5  ───── SW  (Buton) ── INPUT_PULLUP  │
  │   3.3V   ───── VCC  (gerekiyorsa)           │
  │   GND    ───── GND                          │
  │                                             │
  │   (Alternatif — Potansiyometre)             │
  │   GPIO34/A0 ── ORTA PIN                     │
  │   3.3V  ─────── SOL PIN                     │
  │   GND   ─────── SAĞ PIN                     │
  │                                             │
  └─────────────────────────────────────────────┘

  Güç: USB-C / 5V  →  ESP32 3.3V regülatörü üzerinden beslenir.
  OLED I2C çekme dirençleri: SDA & SCL hatlarına 4.7 kΩ (3.3V'a).
